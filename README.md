
# Projet web (HTML/CSS/JS) sur le jeu-vidéo Kingdom Rush

## Prérequis

Avant de commencer, vérifiez que vous avez :
- Node.js + npm
- Python 3

## Installation

Depuis le dossier du projet (`ComplementWeb`), installez les dépendances :

```bash
npm install
```

## Lancer le projet

```bash
npm run dev
```

Cette commande démarre :
- l'API sur `http://localhost:3000`
- le front sur `http://localhost:5500`

Ouvrez ensuite votre navigateur sur :
- `http://localhost:5500`
